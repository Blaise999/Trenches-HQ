# Trenches HQ

P2P Gift Card Trading & Social Platform — Next.js prototype.

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Plus Jakarta Sans + IBM Plex Mono

## Features

- **Feed** — Card-based posts with trade announcements, milestones, likes, comments
- **Marketplace** — P2P gift card listings with search, category filters, seller ratings, escrow badges
- **Messages** — Conversation list with trade/personal badges, full chat with send
- **Profile** — Stats grid, trade history, completion rate
- **Compose** — Post creation modal
- **Notifications** — Activity panel with trade, system, social notifications

All data is pre-populated. Mobile-first design with bottom tab navigation.
